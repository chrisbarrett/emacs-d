# Feature: hex

Hexadecimal file editing with vim-style keybindings.

## Files

| File            | Purpose                    |
| :-------------- | :------------------------- |
| init/init-hex.el | Hexl mode keybindings     |

## Packages

| Package | Source   | Purpose              |
| :------ | :------- | :------------------- |
| hexl    | Built-in | Hex editor mode      |

## Behaviors

### Vim-Style Keybindings

Evil normal state keybindings for hexl-mode:

| Key   | Command                      | Action                     |
| :---- | :--------------------------- | :------------------------- |
| `h`   | `hexl-backward-char`         | Move left                  |
| `l`   | `hexl-forward-char`          | Move right                 |
| `]]`  | `hexl-end-of-1k-page`        | Jump to end of 1K page     |
| `[[`  | `hexl-beginning-of-1k-page`  | Jump to start of 1K page   |
| `$`   | `hexl-end-of-line`           | Jump to end of line        |
| `^`   | `hexl-beginning-of-line`     | Jump to start of line      |
| `0`   | `hexl-beginning-of-line`     | Jump to start of line      |

### Navigation Model

The keybindings follow vim conventions:
- `h`/`l` for character movement (not word movement like in vim text modes)
- `[[`/`]]` for 1K page navigation (analogous to section movement)
- `0`/`^`/`$` for line navigation

## Testable Properties

1. `h` bound to `hexl-backward-char` in hexl-mode normal state
2. `l` bound to `hexl-forward-char` in hexl-mode normal state
3. `]]` bound to `hexl-end-of-1k-page` in hexl-mode normal state
4. `[[` bound to `hexl-beginning-of-1k-page` in hexl-mode normal state
5. `$` bound to `hexl-end-of-line` in hexl-mode normal state
6. `0` and `^` both bound to `hexl-beginning-of-line` in hexl-mode normal state
